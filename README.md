## Deskripsi Proyek

Proyek ini bertujuan membandingkan performa metode CNN From Scratch dan Transfer Learning MobileNetV2 pada klasifikasi citra kucing dan anjing menggunakan dataset Cats vs Dogs dari Kaggle.

## Dataset

Dataset yang digunakan:

Cats vs Dogs Dataset (Kaggle)

https://www.kaggle.com/datasets/tongpython/cat-and-dog

Jumlah kelas:

- Cat (Kucing)
- Dog (Anjing)

## Metode

- CNN From Scratch
- Transfer Learning MobileNetV2

## Hasil Pengujian

### CNN From Scratch

- Accuracy : 75.60%
- Loss : 0.9814

### Transfer Learning MobileNetV2

- Accuracy : 98.67%
- Loss : 0.0424

## Kesimpulan

Berdasarkan hasil eksperimen, Transfer Learning MobileNetV2 menghasilkan performa yang lebih baik dibandingkan CNN From Scratch. Model MobileNetV2 memperoleh akurasi yang lebih tinggi dengan nilai loss yang lebih rendah sehingga lebih efektif digunakan untuk klasifikasi citra kucing dan anjing.
